📂 Tugas Porto

Repository ini berisi tugas portofolio yang dibuat untuk menampilkan hasil pembelajaran dalam pengembangan website atau pemrograman. Proyek ini berfungsi sebagai latihan dalam membuat repository GitHub serta mendokumentasikan proyek dengan baik.

📌 Deskripsi Proyek

Project ini merupakan website portofolio sederhana yang menampilkan informasi tentang diri, proyek, serta kemampuan yang dimiliki oleh pembuat.

Tujuan dari proyek ini adalah:

Melatih penggunaan Git dan GitHub

Membuat website portofolio sederhana

Menyusun dokumentasi proyek melalui README

🚀 Fitur

Beberapa fitur yang tersedia pada project ini:

Halaman Portofolio / Proyek
